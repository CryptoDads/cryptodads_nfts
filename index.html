CryptoDads NFT Metadata & Images

This repository hosts the metadata and images for CryptoDads NFTs.

API Endpoints

Metadata: https://cryptodads.github.io/cryptodads_nfts/metadata/{token_id}
Images: https://cryptodads.github.io/cryptodads_nfts/images/{token_id}.png

Usage

Replace {token_id} with the actual token ID (e.g., 1, 2, 3, etc.)
